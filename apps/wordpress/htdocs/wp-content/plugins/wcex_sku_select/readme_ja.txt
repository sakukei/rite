******************************************
　　　WCEX SKU Select for Welcart
******************************************


WCEX SKU Select はWelcart専用の拡張プラグインです。

WCEX SKU Selectは、SKUをセレクト又はラジオボタン形式で表示する事で、 
これまでSKUごとに表示されていたカートボタンを1つにまとめることができる拡張プラグインです。

ご利用方法の詳細は、オンラインドキュメントをご覧ください。
https://www.welcart.com/docs/SkuSelect/index.html


==============
ご利用方法
==============
【インストール】
管理パネルのプラグインの新規追加画面（プラグインのインストール）から
「アップロード」リンクをクリックして、ダウンロードしたWCEX SKU Select の
圧縮ファイル（[例]wcex_sku_select.1.1.zip）をそのままアップロードして
有効化します。
